# ADR-002 – Escolha de Tecnologias

### Contexto

O sistema precisa de um conjunto de tecnologias que permita o desenvolvimento de uma aplicação robusta, evolutiva e alinhada com a Clean Architecture, garantindo que as regras de negócio permaneçam desacopladas de frameworks e ferramentas específicas.

Além disso, as tecnologias escolhidas devem ser amplamente utilizadas no mercado, facilitando aprendizado, manutenção e futuras evoluções do projeto.

---

### Decisão

As seguintes tecnologias foram escolhidas para o desenvolvimento do sistema:

- **Linguagem:** Java 17
    
- **Framework:** Spring Boot
    
- **Persistência:** JPA (Java Persistence API) com Hibernate
    
- **Banco de Dados:** PostgreSQL
    
- **Build Tool:** Maven
    
- **Testes:** JUnit 5 e Mockito
