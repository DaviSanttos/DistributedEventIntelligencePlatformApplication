# ADR 001 - Escolha do Banco de Dados

## Status
Aceito

## Contexto
Precisamos de um banco que suporte alto volume de escrita
e transações consistentes.

## Decisão
Utilizar PostgreSQL.

## Alternativas Consideradas
- MySQL
- MongoDB

## Consequências
- Melhor suporte a transações
- Maior custo operacional que MongoDB
